## 프로그래밍(네이밍) 표기법
- 프로그래밍 언어에서는 대소문자 구분
- 가독성과 연관
- 변수, 함수, 클래스 등의 이름을 정할 때 여러 개의 영단어를 사용할 경우 각각의 단어를 구분
```C++
// 0. 일반적인 문장 :
unreal engine function variable 
// 1. Snake Case :
unreal_engine_function_variable 
// 2. Kebab Case :
unreal-engine-function-variable 
// 3. Camel Case :
unrealEngineFunctionVariable 
// 4. Pascal Case :
UnrealEngineFunctionVariable
```



## 배열
- 배열 변수 선언 시 크기 인덱스는 변수로 지정할 수 없음
- C++ 컴파일 언어 => 컴파일하는 시간에 배열 크기가 결정

    - 구조체 : 변수만으로 집합체 구성
